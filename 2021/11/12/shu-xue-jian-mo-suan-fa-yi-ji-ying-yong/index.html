线性规划（matlab）

适用问题

多变量的不等关系和相等关系，通过建立坐标系利用限制方程对区域进行规划限制

  • 投资类问题求解最优收益率的问题（考虑风险率，收益率以及交易费用等等）
  • 工厂生产不同成本不同价值商品的最优收入
  • 飞机货运平衡问题（前，后，中如何保持飞机平稳）
  • 其他最大值最优解问题

基本原理

分别对不同的变量进行不等式的构造，得到问题要求的各个变量的最小值or最大值。

将各个变量的不等公式最优解带入到目标方程中。

整数规划（Lingo）

基本原理：

将部分变量转化成整数，达到简化运算的目的。具体分为01整形规划、蒙特卡洛算法以及匈牙利算法。

01整形规划：利用某些变量只有两种变化的特性，对限制条件进行增添，从而简化运算。

蒙特卡洛算法：利用计算机进行模拟，适合数据量较大的规划问题

适用问题:

  • 指派问题（01规划，构建矩阵）
  • 不规则图形面积求解（蒙特卡洛算法）
  • 混合型问题，考虑整数以及其他多重变量。
  • 最优分配问题（建最少的小学覆盖最大的范围）

插值与拟合

基本原理

插值：如何求解一个确定的函数？显然不能直接通过实验进行推理，因此可以将函数分解成无穷多的小段函数，再确定局部与整体的关系吧！

拟合：如何求解众多数据点之间的函数关系呢？拟合成具体的函数吧！

  • 机床机件加工
  • 判断函数最高点最低点的数值

数理统计

基本原理

以概率论为基础，从少数的数据判断整体的变化，少量数据确定整体的全部特征。

基本：参数估计、假设检验

前提条件：求解置信区间：

经验分布函数：
坐标纸上绘制离散的数据，通过阶梯式的方法连接；

Q-Q图，经验函数（检验拟合优度）

计算步骤：

1.将x1,x2,…,xn依大小顺序排列成x1≤x2≤…≤xn

2.取y=1/F((i-1/2)/n) ,i =1 ,2,…,n。
3．将(yi,xi ) ,i = 1 ,2,…,n,这n个点画在直角坐标图上。
4.如果这n个点看起来呈一条45°角的直线,从(0,0)到(1,1)分布,则x1,x2…xn拟合分布F(x)的效果很好。

柯尔莫哥洛夫检验：
对曲线的拟合效果进行直接的分析，分析构造出的函数值与离散数据的差值的极限，区别于之前的分析离散数据的递推公式。

秩和检验：
1.将两个样本混合起来,按照数值大小统一编序,由小到大,每个数据对应的序数称为秩。

2.计算取自总体X的样本所对应的秩之和,用T表示。（取最小）
3．根据n ,n,与水平α,查秩和检验表,得秩和下限T与上限 T。（不同条件下自变量对应空白数据的秩数之和）
4．如果T≤T或T≥T,,则否定假设H,认为X,Y两总体分布有显著差异

。否则认为X、Y两总体分布在水平α下无显著差异。

Bootstrap方法

对极少的数据进行抽样处理（如放回抽样），得到更多样本后（n≥1000）,对各组数据的中位数进行标准差处理，从而分析精度。

适用问题

  • 求解置信区间
  • 检验是否满足正态分布、泊松分布
  • 两种分布的比较，判断是否一致
  • 极少数据估测分布情况
  • 对未知参数通过公式进行估算

时间序列（趋势预测）

基本原理

将预测对象按照时间顺序排列，根据现有和过去的时间序列特点规律，推断今后变化趋势、变化规律。

如何实现？

一次：选定合适的移动区间值N，从区间左端向右端一步一步推算出第N+2的自变量点的预测值（移动区间内对数据求平均值），最后得到想要的结果。

二次：在一次进行计算的基础上，利用计算得到的数据再次进行平均移动计算。

检验方法：预测标准误差公式：

一次指数平滑

%原始数据cancerdata.txt可在网上下载,数据中的B替换成1,M替换成-1,X替换成⒉,删
%除了分割符”,替换后的数据命名成 cancerdata2.txt
clc,clear
a = load( 'cancerdata2.txt');
a( :,1)=[];%删除第一列病例号
gind = find(a(:,1)==1);%读出良性肿瘤的序号
bind = find( a( :,1）==-1);%读出恶性肿瘤的序号
training =a([1:500],[2:end]);%提出已知样本点的数据
training =training';%为了进行数据标准化,这里进行了转置
[ train,ps] =mapstd( training); %已分类数据标准化
group(gind)=l ; group( bind) = -1; %已知样本点的类别标号
group = group'; %转换成列向量
xa0 = a([501:569],[2:end]);% 提出待分类数据
xa = xa0'; 
xa = mapstd( 'apply',xa,ps);%待分类数据标准化
s = svmtrain(train',group，'Method' ,' sMo' , 'Kernel_Function' , ' quadratic')%使用序列最小化方法训练支持向量机的分类器,如果使用二次规划的方法训练支持向量机则无法求解
sv_index = s.SupportVectorIndices' % 返回支持向量的标号
beta = s.Alpha'%返回分类函数的权系数
b=s.Bias %返回分类函数的常数项
mean_and_std_trans = s.ScaleData %第1行返回的是已知样本点均值向量的相反数,第2行返回的是标准差向量的倒数
check = svmclassify(s,train');%验证已知样本点
err_rate =1 - sum( group = = check )/Alength( group)%计算错判率
solution = svmclassify( s,xa');%进行待判样本点分类
solution = solution'
sg = find( solution = =1）8求待判样本点中的良性编号
sb = find(solution = = -1）%求待判样本点中的恶性编号
